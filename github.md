# Push Project to GitHub Main Branch

## Prerequisites
- Git installed on your system
- GitHub account created
- Repository created on GitHub

## Steps

### 1. Initialize Git Repository
```bash
git init
```

### 2. Add Remote Origin
```bash
git remote add origin https://github.com/yourusername/momsStore2.git
```

### 3. Add All Files
```bash
git add .
```

### 4. Create Initial Commit
```bash
git commit -m "Initial commit"
```

### 5. Set Main Branch
```bash
git branch -M main
```

### 6. Push to GitHub
```bash
git push -u origin main
```

## Additional Commands

### Check Git Status
```bash
git status
```

### View Remote URLs
```bash
git remote -v
```

### Add Specific Files
```bash
git add filename.ext
```

### Commit with Message
```bash
git commit -m "Your commit message"
```
